# SAEIQRS-T-virus-propagation
Numerical simulations, agent-based model, and optimization routines for the SAEIQRS-T computer virus propagation model.

**Compartmental models for the propagation of computer viruses: analysis and simulation**

The project studies the propagation of computer viruses in networks using an extended compartmental model called **SAEIQRS-T**, which includes the following compartments:

- **S**: Susceptible devices
- **A**: Antidote or protected devices
- **E**: Exposed devices
- **I**: Infected devices
- **Q**: Quarantined devices
- **R**: Recovered devices
- **T**: Transmission nodes

The repository includes MATLAB scripts for symbolic analysis and numerical simulations, as well as a C# agent-based simulation developed with Avalonia. The agent-based model also includes a Grey Wolf Optimizer implementation to identify critical transmission nodes.
